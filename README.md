# ai-localization-website

## How to Run

1.  **Ensure Python is installed:** If you don't have Python installed, download and install it from [python.org](https://www.python.org/downloads/).
2.  **Unzip the `ai website.zip` file.** This will create a `website` directory.
3.  **Open a terminal or command prompt.**
4.  **Navigate to the `website` directory:**
    ```bash
    cd website
    ```
5.  **Start the Python HTTP server:**
    ```bash
    python -m http.server
    ```
    (If you have Python 2, the command might be `python -m SimpleHTTPServer`)
6.  **Open your web browser** (like Chrome, Firefox, Safari, or Edge).
7.  **Go to the address:** `http://localhost:8000`

You should now see the AI localization website.
